# Retail-Inventory-Analytics-EDA-Auto-Reorder-System

• Performed Exploratory Data Analysis (EDA) using Python on 1,000+ records across 25 columns to clean data and analyze sales, profit, product categories, and regional performance.

• Translated Python analysis into 3 interactive Power BI dashboards (Performance, Inventory, Customer Insights) displaying sales and profit KPIs, regional trends, and monthly performance.

• Designed dashboards with 10+ visuals including charts, maps, and filters to enable quick data-driven business insights.
